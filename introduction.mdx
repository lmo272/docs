# Welcome to Lynks AI External API

Lynks AI provides a powerful API for enhancing product data, specifically tailored for Manor & Galaxus. Our API offers a range of features to optimize product information, categorize products, and extract attributes.

## Key Features

- **Product Name Optimization**: Improve product names for better clarity and searchability.
- **Product Description Optimization**: Enhance product descriptions with adjustable tone, style, and SEO optimization.
- **Product Categorization**: Automatically categorize products for specific sales channels.
- **Attribute Extraction**: Extract relevant attributes from product descriptions based on category requirements.
- **Data Retrieval**: Access categories, attributes, and values for Manor & Galaxus.

## Getting Started

To start using the Lynks AI External API, you'll need to:

1. Obtain API credentials (API Key and OpenAI Key)
2. Familiarize yourself with the available endpoints and their parameters
3. Make API requests to enhance your product data
